# Snowcrash

## What is snowcrash ?

Snowcrash is the first security project.

You will have to find the flags for each level, allowing you to connect to the next one

[French Subject](snow_crash.fr.pdf)

[Iso download](https://projects.intra.42.fr/uploads/document/document/306/SnowCrash.iso)

It is recommended to use ssh ! if you do so, use:

```
ssh -p 4242 levelXX@ip_of_my_vm
```

To connect to the first level, you can use use the following:

```
login:    level00
password: level00
```
